## Algorithm
1. หาเซตของตัวอักษรที่ไม่ซ้ำกันใน string
2. ทดลองจับคู่ตัวอักษรทีละ 2 ตัว (combination)
3. สำหรับแต่ละคู่:
   - กรอง string ให้เหลือเฉพาะ 2 ตัวอักษรนั้น
   - ตรวจสอบว่า string ที่ได้มีรูปแบบสลับกันหรือไม่
4. ถ้าเป็น alternating:
   - บันทึกความยาวของ string
5. คืนค่าความยาวที่มากที่สุด
6. ถ้าไม่มีคู่ใดผ่านเงื่อนไข คืนค่า 0
