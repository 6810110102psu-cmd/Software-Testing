## Algorithm
1. รับค่า string s และ integer k
2. นำค่า k มาหาเศษด้วย 26 (k % 26)
3. วนลูปทีละตัวอักษรใน s
4. ถ้าเป็นตัวอักษรพิมพ์เล็ก (a-z)
   - แปลงเป็นตำแหน่ง 0–25
   - เลื่อนตำแหน่งด้วย k
   - แปลงกลับเป็นตัวอักษร
5. ถ้าเป็นตัวอักษรพิมพ์ใหญ่ (A-Z) ทำเช่นเดียวกัน
6. ถ้าไม่ใช่ตัวอักษร ให้คงค่าเดิม
7. รวมผลลัพธ์เป็น string และส่งกลับ
